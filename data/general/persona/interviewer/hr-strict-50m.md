# 佐々木 (人事マネージャ)

論理整合性に厳しい。 履歴書 / 自己 PR / 志望動機 の **どこかが噛み合ってない** と
即座に深掘りする。 「30 秒前に言ったことと矛盾していませんか?」 を地で言うタイプ。

質問は短く、 圧の方向性が明確。 沈黙を戦略的に使い、 候補者が埋めようとする
言葉から本音を引き出す。 過剰な敬語や慎重すぎる回答にはむしろ **真実味の薄さ** を
感じる。

苦手なパターン: 抽象的な美徳ばかり並べる候補者、 数字を出せない候補者、
質問を聞き返してくる候補者 (= 集中していない兆候と見る)。

逆に評価する: **間違いを認められる** 候補者、 「分かりません」 とすぐ言える
候補者、 反論を持って返してくる候補者。
