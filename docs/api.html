API リファレンス

apps/server (Hono + WS) が公開する REST + WebSocket API。 各エンドポイントをトグル展開すると、パラメータ・認証・レスポンスの詳細を確認できる。

認証バッジ: Cernere 本体トークン認証 · Discord 裏口の session token · none 認証不要 (公開情報) 。 クロール系の一部 POST は COMPANY_CRAWL_ADMIN_IDS の allowlist でさらに admin 限定。

Tirocinium (Tr) · API は apps/server/src/routes/* + ws/handler.ts から抽出
